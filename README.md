WinForms Tennis Court Reservation System
This application is a comprehensive solution for managing user authentication, personal profiles, image uploads, and tennis court reservations. Developed with Windows Forms and SQL Server, it provides an intuitive interface for users to book tennis courts, manage reservations, and view personal information.

Features
User Authentication: Secure login system with username and password.
Personalized Welcome: Displays user information including username, email, gender, and profile picture.
Image Upload: Allows users to upload a profile picture.
Court Booking: Interface for users to check the availability of tennis courts and book them.
Reservation Management: Enables users to view and manage their existing reservations.
Getting Started
These instructions will guide you to get a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites
What you need to install the software:

Microsoft SQL Server
.NET Framework 4.7.2 or higher
Installing
A step by step series of examples that tell you how to get a development env running:

Clone the repository to your local machine.
Open the solution file in Visual Studio.
Restore all NuGet packages.
Set up the database using the script provided in the SQL folder.
Build and run the application from Visual Studio.
Usage
To use the application, start it from Visual Studio or the compiled executable. Log in with your user credentials. Once logged in, you can upload a profile picture, book tennis courts, and manage your reservations through the user interface.
